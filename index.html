CSS Cheat Sheet

1. Fonts and Text Styling

+
  • Font Sizes: px, pt, em, rem
  • Font Weights: light, bold, lighter, bolder, 100-900
  • Font Families: serif, sans-serif
  • Text Properties: text-align, text-decoration
  • Lists: list-style-type

2. Colors

+
  • Hexadecimal: #RRGGBB
  • Hex with Transparency: #RRGGBBAA

3. Box Model

+
  • Margin: Outside spacing
  • Padding: Inside spacing
  • Border: Defines box boundary
  • Box-sizing: border-box includes padding and border in width/height

4. Units

+
  • Absolute: px, pt, in
  • Relative: %, em, rem, vw, vh
  • Auto: size based on content
  • Fit-content: adjusts size to fit content

5. Selectors

+
  • Grouping: "," for shared styles
  • Descending: " " for all descendants
  • Direct children: ">"
  • Pseudo-selectors: :hover, :active, :focus
  • Attribute-based: [attribute=value]

6. Positioning

+
  • Static: Default, no positioning
  • Relative: Adjust relative to normal position
  • Absolute: Relative to nearest positioned ancestor
  • Fixed: Relative to viewport
  • Sticky: Behaves like relative until scroll threshold

7. Flexbox

+
  • Flex-direction: row, column
  • Flex-wrap: wrap for responsiveness
  • Justify-content: Align on main axis
  • Align-items: Align on cross axis
  • Gap: Space between items

8. Grid

+
  • Define grid with template-rows and template-columns
  • Place items with grid-row and grid-column
  • Span items across rows/columns
  • Gap: Space between grid items

9. Transitions and Animations

+
  • Transitions: property duration timing-function delay
  • Animations: Use @keyframes for complex effects

10. Responsiveness

+
  • Media Queries: @media (max-width: x)
  • Flexbox and Grid for adaptive layouts
  • Use relative units (%, rem, vw, vh)

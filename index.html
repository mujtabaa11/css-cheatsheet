CSS Cheat Sheet

1. Fonts and Text Styling

+

  • 1.1 Font Sizes: px, pt, em, rem

    +

    px: Absolute unit representing pixels.

    pt: Points, commonly used in print.

    em: Relative unit based on the font size of the parent.

    rem: Relative to the root element's font size.

  • 1.2 Font Weights: light, bold, lighter, bolder, 100-900

    +

    Font weight determines the thickness of characters. Options include:

    • light: Lighter than normal.
    • bold: Thicker than normal.
    • 100-900: Numeric weights, where 400 is normal, 700 is bold.

  • 1.3 Font Families: serif, sans-serif

    +

    Font families define the appearance of text. Common examples:

    • serif: Fonts with small lines at the ends of characters (e.g., Times New Roman).
    • sans-serif: Clean fonts without lines at the ends (e.g., Arial).

  • 1.4 Text Properties: text-align, text-decoration

    +
    • text-align: Align text left, right, center, or justify.
    • text-decoration: Underline, overline, line-through, none.

  • 1.5 Lists: list-style-type

    +

    Define the marker style for lists:

    • disc: Default bullet style.
    • circle: Hollow circles.
    • square: Solid squares.
    • none: No marker.

2. Colors

+

  • 2.1 Hexadecimal: #RRGGBB

    +

    Defines colors using hexadecimal codes, where:

    • RR: Red component (00 to FF).
    • GG: Green component (00 to FF).
    • BB: Blue component (00 to FF).

  • 2.2 Transparency: #RRGGBBAA

    +

    Add transparency with the alpha channel:

    • AA: Alpha component (00 to FF, where FF is opaque).

  • 2.3 CSS Functions: rgb(), rgba(), hsl(), hsla()

    +

    Define colors with functional notations:

    • rgb(red, green, blue): Values range from 0 to 255.
    • rgba(red, green, blue, alpha): Adds transparency.
    • hsl(hue, saturation, lightness): Uses the HSL color model.
    • hsla(hue, saturation, lightness, alpha): Adds transparency to HSL.

3. Box Model

+

  • 3.1 Margins: margin

    +

    Defines space outside the element's border:

    • margin-top: Space above the element.
    • margin-right: Space to the right.
    • margin-bottom: Space below.
    • margin-left: Space to the left.

  • 3.2 Padding: padding

    +

    Defines space inside the element, between content and border:

    • padding-top: Padding above the content.
    • padding-right: Padding to the right.
    • padding-bottom: Padding below.
    • padding-left: Padding to the left.

  • 3.3 Borders: border

    +

    Defines the boundary of an element:

    • border-width: Thickness of the border.
    • border-style: Solid, dotted, dashed, double, etc.
    • border-color: Sets the border's color.

  • 3.4 Box Sizing: box-sizing

    +

    Controls how the total width and height of the element is calculated:

    • content-box: Default, width and height include only the content.
    • border-box: Includes padding and border in the width and height.

4. Positioning

+

  • 4.1 Static: static

    +

    Default position; the element is positioned according to the normal document flow.

  • 4.2 Relative: relative

    +

    The element is positioned relative to its normal position.

    • top: Moves the element up or down.
    • left: Moves the element left or right.

  • 4.3 Absolute: absolute

    +

    The element is positioned relative to the nearest positioned ancestor.

  • 4.4 Fixed: fixed

    +

    The element is positioned relative to the viewport and doesn't move when scrolling.

  • 4.5 Sticky: sticky

    +

    The element toggles between relative and fixed positioning based on scroll position.

5. Flexbox

+

  • 5.1 Display: flex

    +

    Activates flexbox for the container:

    • flex-direction: Row (default), column, row-reverse, column-reverse.

  • 5.2 Justify Content: justify-content

    +

    Defines alignment along the main axis:

    • flex-start: Align items to the start.
    • center: Center items.
    • space-between: Equal space between items.

  • 5.3 Align Items: align-items

    +

    Align items along the cross axis:

    • stretch: Default, items stretch to fill.
    • center: Center items.

  • 5.4 Flex Wrap: flex-wrap

    +

    Determines whether items should wrap to the next line:

    • nowrap: No wrapping (default).
    • wrap: Items wrap onto multiple lines.

6. Grid

+

  • 6.1 Display: grid

    +

    Activates grid layout for the container:

    • grid-template-columns: Defines the column structure, e.g., repeat(3, 1fr).
    • grid-template-rows: Defines the row structure, e.g., 100px auto 100px.

  • 6.2 Grid Gaps: gap

    +

    Sets the spacing between grid items:

    • row-gap: Space between rows.
    • column-gap: Space between columns.

  • 6.3 Grid Areas: grid-area

    +

    Assigns a grid item to a specific area:

    • grid-area: Uses named areas or row/column span.
    • grid-template-areas: Defines the layout structure using area names.

  • 6.4 Grid Alignment: justify-items & align-items

    +

    Aligns grid items within their cells:

    • justify-items: Aligns items along the row axis.
    • align-items: Aligns items along the column axis.

7. Transitions and Animations

+

  • 7.1 Transitions: transition

    +

    Adds smooth changes between property values:

    • transition-property: Specifies the property to transition.
    • transition-duration: Sets how long the transition lasts.
    • transition-timing-function: Defines the speed curve (ease, linear, etc.).

  • 7.2 Animations: animation

    +

    Creates keyframe-based animations:

    • animation-name: Name of the keyframe animation.
    • animation-duration: How long the animation runs.
    • animation-iteration-count: Number of times it repeats.
    • animation-timing-function: Speed curve of the animation.

8. Responsive Design

+

  • 8.1 Media Queries: @media

    +

    Applies styles based on screen size or device features:

    • max-width: Applies when the screen width is less than or equal to the specified value.
    • min-width: Applies when the screen width is greater than or equal to the specified value.

  • 8.2 Viewport Units: vw, vh

    +

    Units based on viewport size:

    • vw: 1% of the viewport width.
    • vh: 1% of the viewport height.

  • 8.3 Flexibility: flex and grid

    +

    Uses flexible layouts to adjust content:

    • Flexbox: Responsive layout with flexible alignment.
    • Grid: Complex layouts with defined grid areas.

  • 8.4 Relative Units: em, rem

    +

    Units relative to other measurements:

    • em: Relative to the font size of the parent element.
    • rem: Relative to the root font size.
